# Movie App API

Backend API for a Movie App similar to [this app](https://react-movies-list.netlify.app/).
Stack/tools includes:

- Java (23 SDK)
- Spring Boot
- Spring Security (OAuth2)
- Spring Data MongoDB
- MongoDB
- Docker
- Lombok
- Model Mapper
- Testcontainers
- Swagger
- Maven

## Swagger UI

`http://localhost:8080/swagger-ui/index.html`
